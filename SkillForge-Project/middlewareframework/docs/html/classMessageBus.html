Detailed Description

Thread-safe implementation of IMessageBus using atomic shared_ptr for subscribers.

Supports multiple subscribers and allows publishing messages asynchronously. Internally uses a shared_ptr to a vector of callbacks to allow lock-free iteration during publish, while subscribe updates are done atomically.

Publish a message to all subscribers.

Iterates over all subscribed callbacks and invokes them with the given message.

Parameters
msgThe message to be published.

This method atomically loads the current list of subscribers and invokes each registered callback with the provided message. If no subscribers are present, the function returns immediately.

void MessageBus::subscribe ( Callback  callback)
overridevirtual

Subscribe a callback function to receive published messages.

Subscribe a new callback to the message bus.

The callback will be invoked each time a message is published.

Parameters
callbackFunction to call when a message is published.

This method uses lock-free compare-and-swap to safely update the list of subscribers in a thread-safe manner. Each new subscriber is added to a copy of the current list, which is then atomically swapped in.

Parameters
callbackThe callback function to be invoked on message publish.
