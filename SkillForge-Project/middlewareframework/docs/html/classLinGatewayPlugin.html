Detailed Description

Sensor plugin for creating LIN gateway sensors.

Parses JSON configuration to set up polling intervals, frame schedules, and signal mappings. Creates a LinGatewaySensor instance connected to a simulated LIN bus.

Create a LinGatewaySensor instance based on JSON configuration.

Create a LIN gateway sensor instance.

Parses configuration for polling interval, schedule, and signals, then constructs a LinGatewaySensor with a simulated LIN bus and driver.

Parameters
busReference to the application's message bus.
configJSON configuration.
Returns
std::unique_ptr<ISensor> Constructed sensor instance.

Parses the provided JSON configuration to extract:

  • polling interval
  • frame schedule
  • signal mapping for normalization

If signals are not provided, a default demo mapping is used.
