Detailed Description

MQTT Adapter plugin bridging MessageBus to MQTT broker.

Connects to a broker using TLS, subscribes to a topic, and publishes messages from the MessageBus to MQTT. Also consumes MQTT messages and republishes them to the MessageBus.

Implements the IPlugin interface for dynamic plugin loading.
